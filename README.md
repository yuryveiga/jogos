## Como usar

Para visualizar o site localmente:
1. Abra o arquivo `index.html` em seu navegador.
2. Ou use um servidor local (ex: `python3 -m http.server`).

## Publicação no GitHub Pages

Para publicar este site no GitHub Pages:
1. Crie um novo repositório no GitHub.
2. Faça o upload de todos os arquivos desta pasta para o repositório.
3. Vá em **Settings** > **Pages**.
4. Em **Build and deployment**, selecione a branch `main` (ou `master`) e a pasta `/ (root)`.
5. Clique em **Save**.

O site estará disponível em `https://seu-usuario.github.io/nome-do-repositorio/`.
